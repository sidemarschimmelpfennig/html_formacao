O < !DOCTYPE html > declaração define que este documento é um documento HTML5.
O elemento < html > é o elemento raiz de uma página HTML."
O elemento < head > contém informações meta sobre a página HTML."
O elemento < title > especifica um título para a página HTML (que é exibido na barra de título do navegador ou na aba da página)."
O elemnto < body > define o corpo do documento e é um container para todas as informaçoes visiveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.
O elemento < h1 > define um titulo grande
O elemento < p > define um paragrafo
